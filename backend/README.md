# Auteur AI Backend (Spring Boot)

Java 21 + Spring Boot 3.2 backend API.

## Setup

Requirements:

- Java 21 LTS
- Maven 3.9+

## Development

```bash
mvn spring-boot:run
```

## Build

```bash
mvn clean package
```

The JAR file will be generated in `target/`.
